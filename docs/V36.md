# V36 Command Lifecycle Stabilization

This package adds a centralized local operations store at:

`apps/web/lib/metis-v36-store.ts`

It provides:
- event persistence
- personnel persistence
- audit persistence
- communication thread persistence
- assignment reconciliation
- resolved-event release behavior
- compatibility with prior localStorage keys

The next implementation step should migrate pages to consume this centralized store instead of duplicating localStorage logic.
