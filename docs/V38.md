# V38 Workload Intelligence

V38 adds a decision-support layer for local operational state.

## Scope
- Personnel workload scoring
- Best assignment candidate recommendation
- Assignment balancing
- Coverage gap alerts
- Overloaded personnel warnings
- Staffing health metrics
- Event-to-personnel ratio

## Test
1. Create events.
2. Assign personnel to some events.
3. Open `/intelligence`.
4. Confirm recommendations change as assignments change.
5. Confirm unassigned events appear as coverage gaps.
